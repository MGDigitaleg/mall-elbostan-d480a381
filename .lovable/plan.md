

## تحسين قائمة التنقل (Header Menu Sheet)

تحسين شامل لمكوّن `HeaderMenuSheet.tsx` لرفع جودة العرض البصري وتنظيم المحتوى بشكل أكثر هدوءاً واحترافية، مع توحيدها مع نظام الألوان الخاص بالمول (Deep Navy + Ivory + Primary Blue).

---

### المشاكل الحالية الواضحة في الصور
1. **الـ Header الداخلي**: اللوجو مقصوص من الأعلى بسبب padding غير كافٍ، والخلفية الداكنة تبدو منفصلة عن باقي القائمة بحدة بصرية مزعجة.
2. **شريط "الصفحة الحالية"**: يبدو معلّقاً ومنفصلاً، وحجمه الصغير يجعله يبدو كـ tag تائه.
3. **الأيقونات**: كل العناصر في قسم "المول" تستخدم نفس الأيقونة (`Building2`) — يفقد التمييز البصري السريع.
4. **CTA السفلي البرتقالي**: قوي جداً ويتنافس مع الـ "أدر واربح" الأزرق في الأعلى — يكسر الهيرارشي.
5. **Theme Toggle**: تصميمه كصف منفصل لا يتسق مع باقي العناصر.
6. **Quick Actions**: زر "الخريطة" بـ glassmorphism خفيف مع زر "أدر واربح" بـ gradient أزرق — اختلاف بصري حاد بدون مبرر.

---

### الحلول المقترحة

#### 1. تحسين الـ Header الداخلي
- زيادة الـ padding العلوي ليتنفس اللوجو (`pt-8` بدل `pt-6`).
- استبدال gradient الـ Navy بتدرج أكثر نعومة مع overlay خفيف.
- توسيط محاذاة اللوجو + جعل حجمه أصغر ومتناسق (`max-w-[110px]`).
- إضافة فاصل بصري ناعم بين الـ Header الداكن والمحتوى الفاتح (انحناء `rounded-b-3xl` بدل قاطع).

#### 2. إعادة تصميم Quick Actions
- توحيد الستايل: زرّان متماثلان بنفس البنية (icon + label) مع تمييز فقط للـ primary action ("أدر واربح").
- "الخريطة" → outline ناعم بحدود ذهبية شفافة.
- "أدر واربح" → solid blue gradient (يبقى الـ accent الوحيد القوي في الـ Header).
- زيادة الارتفاع لـ `h-12` وتحسين tap target.

#### 3. شريط الصفحة الحالية
- دمجه داخل الـ Header الداكن نفسه كـ `breadcrumb-style indicator` (مثل: "أنت في: الخريطة التفاعلية") بلون فاتح خفيف على خلفية شفافة.
- إزالته كعنصر مستقل عائم.

#### 4. تنويع الأيقونات لقسم "المول"
- الرئيسية → `Home`
- عن البستان → `Info`
- صدى السوق → `Sparkles` أو `Radio`
- دليل المحلات → `Store` (يبقى)
- منتجات المحلات → `ShoppingBag` (يبقى)
- الخريطة التفاعلية → `Map` (يبقى)
- العروض اليومية → `Tag` (يبقى)

#### 5. تحسين عناصر الـ Nav
- زيادة الـ vertical padding قليلاً (`py-3` بدل `py-2.5`) لراحة التتبّع.
- إضافة hover state أوضح (transition للخلفية).
- العنصر النشط: شريط جانبي رفيع `border-r-2` بدل النقطة الصغيرة.
- عناوين الأقسام: تكبير قليل + حد فاصل أنعم.

#### 6. منطقة الـ CTA السفلية — إعادة الترتيب
**الترتيب الجديد (من فوق لتحت):**
- **Theme Toggle**: تصميم أصغر وأكثر نعومة كـ `inline pill` مع icon فقط.
- **CTA رئيسي**: "تواصل معنا" بستايل outline أنيق (يصبح هو الأبرز هنا لأن "أدر واربح" موجود فوق).
- **CTA ثانوي**: "استفسر عن الوحدات المتاحة" — يتم نقله للـ Quick Actions بحجم أصغر، أو إزالته من السفل تماماً وإبقاؤه كـ link نصي خفيف.

**البديل الأنظف**: إزالة الـ CTA البرتقالي من الـ Sheet كلياً (لأن "أدر واربح" + "الخريطة" كافيان في Quick Actions، والـ "تواصل معنا" كافٍ في الأسفل). هذا يقلل الضوضاء البصرية ويحترم قاعدة تقليل الكثافة.

#### 7. تحسين theme toggle
- استبداله بـ icon button بسيط في رأس الـ Sheet (بجانب زر الإغلاق) بدل صف كامل في الأسفل.
- يوفر مساحة ويبدو أكثر احترافية.
