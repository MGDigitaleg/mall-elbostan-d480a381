
# إعادة تصميم صفحة العروض (/daily-deals)

## التشخيص الحالي

1. **الهيرو طويل جدًا** ويأخذ كامل الشاشة الأولى (عنوان + وصف + بطاقة عدّاد كاملة + ٣ بطاقات إحصاءات).
2. **سكشن «مختارات سريعة من العروض الحالية»** (السطور 171–205) ضخم جدًا: بطاقة عرض كبيرة + بطاقة جانبية «حالة الصفحة الآن» نصّية بلا قيمة فعلية للمستخدم — مكرّر مع الشبكة الأسفل.
3. **صور العروض** تُعرض داخل `aspect-[16/10]` بـ `object-cover` فقط، بدون معالجة احترافية (بدون خلفية، بدون شارة خصم متحركة، بدون hover zoom واضح، الصور تُقصّ وتفقد التفاصيل).
4. **تكرار بصري**: العرض الرئيسي يظهر في «المختارات» ثم لا يظهر في الشبكة (مستثنى)، فيشعر المستخدم بانقسام غير مبرر.
5. **سكشن «استكشف كل المحلات المشاركة»** (CTA منفصل) يضيف ارتفاعًا بدون قيمة جديدة.

---

## الفكرة الجديدة

استبدال «سكشن المعاينات الضخم» بفكرة **«Spotlight Strip»** أفقي مدمج (شريط أبرز ٣ عروض)، ودمج العدّاد داخل الهيرو بشكل أصغر، وتوحيد كل العروض في شبكة واحدة كثيفة قابلة للفلترة.

### البنية الجديدة للصفحة

```text
┌──────────────────────────────────────────────────────┐
│ HERO مضغوط (py-5)                                    │
│  ├─ عنوان + وصف سطر واحد                              │
│  ├─ شريط إحصائيات أفقي رفيع (inline pills)           │
│  └─ عدّاد سطر واحد + شارة "افتتاح تجريبي"             │
├──────────────────────────────────────────────────────┤
│ SPOTLIGHT STRIP — أبرز ٣ عروض (carousel أفقي)        │
│  بطاقات أعرض، صورة كبيرة جانبية + سعر + CTA          │
├──────────────────────────────────────────────────────┤
│ شريط فلاتر: [كل المحلات] [محل ١] [محل ٢] ...         │
│              + ترتيب (الأحدث/الأقوى/الأقرب لانتهاء)   │
├──────────────────────────────────────────────────────┤
│ شبكة موحّدة كثيفة (sm:2 / lg:3 / xl:4)               │
│  بطاقات OpeningOfferCard المحسّنة                     │
└──────────────────────────────────────────────────────┘
```

## التغييرات التفصيلية

### 1. تقليص الهيرو (`DailyDeals.tsx` السطور 128–168)
- إزالة الجريد ثنائي العمود؛ الهيرو يصبح صف عمودي مضغوط `py-5 md:py-6`.
- العنوان: تقليل إلى `text-[1.25rem] md:text-[1.6rem]`.
- الوصف: سطر واحد فقط (حذف النسخة الطويلة).
- الإحصائيات: تحويل `OfferMetaLine` الثلاثة إلى **شريط inline pills** أفقي صغير (`محلات: 5 · عروض: 23 · مفتوح الآن: 3`) بدلاً من بطاقات منفصلة.
- العدّاد: تحويل بطاقة العدّاد الكبيرة إلى سطر مدمج داخل الهيرو نفسه (`CountdownTimer compact` بدون الحاوية الإضافية والوصف).

**النتيجة:** ارتفاع الهيرو ينخفض من ~420px إلى ~180px.

### 2. حذف سكشن «مختارات سريعة» الكامل (السطور 171–205)
- يُحذف بالكامل ويستبدل بـ **Spotlight Strip** جديد.
- حذف بطاقة «حالة الصفحة الآن» النصّية (لا قيمة عملية).

### 3. مكوّن جديد: `<OfferSpotlightStrip />`
- يعرض أعلى ٣ عروض (مميّزة + الأقوى خصمًا).
- على الديسكتوب: ٣ بطاقات أفقية بصورة جانبية كبيرة (`grid-cols-3`, ارتفاع 200px).
- على الموبايل: scroll أفقي (`overflow-x-auto snap-x`).
- كل بطاقة: صورة 40% + معلومات 60% (شعار المحل، العنوان، السعر القديم/الجديد، شارة خصم بارزة، زر «اذهب للعرض»).
- لا يستثني العروض من الشبكة (يظهرون مرة واحدة في Spotlight ومرة في الشبكة كاملة) — لتجنّب التكرار سنضيف badge «⭐ مميّز» على البطاقات في الشبكة بدل إخفائها… أو نختار إخفاءها من الشبكة فقط (سنختار: **إخفاءها من الشبكة** للحفاظ على الكثافة).

### 4. تحسين عرض صور العروض (`OpeningOfferCard.tsx`)
- **خلفية احترافية للصورة**: استبدال الخلفية الرمادية `bg-muted/30` بـ gradient ناعم (`bg-gradient-to-br from-secondary/40 to-muted/20`) لإبراز المنتج.
- **`object-contain` بدل `object-cover`** للصور لمنع قص المنتج (مع padding داخلي `p-3` للصورة)، مع الإبقاء على `object-cover` لو الصورة عريضة (lifestyle). الحل: `object-contain` افتراضيًا — أنسب لصور المنتجات التقنية.
- **شارة خصم احترافية**: تحريك شارة الخصم إلى أعلى-يسار الصورة بتصميم badge مائل بلون orange مميّز مع ظل (`shadow-lg`) وحجم أكبر.
- **Hover effect محسّن**: `scale-[1.05]` مع `transition-transform duration-700` + إضافة overlay طفيف عند hover.
- **شعار المحل**: نقل الـ overlay السفلي ليكون أصغر وأنظف (شريط شفاف رفيع بدل بطاقة كاملة).
- **استخدام `optimizeImageUrl`** من `src/lib/imageUtils.ts` لتحسين جودة الصور تلقائيًا.

### 5. توحيد الشبكة وحذف سكشن «استكشف المحلات»
- حذف بلوك «استكشف كل المحلات المشاركة» (السطور 299–313) — مكرّر مع شريط الفلاتر.
- توحيد فرعي `!isExpired` و `isExpired`: شبكة واحدة بنفس التصميم في الحالتين، الفرق فقط في عنوان السكشن.
- زر «عرض المزيد» يبقى لكن يصبح يعرض دفعات من ٨ بدلاً من الكل.

### 6. ترتيب وفلاتر داخل شريط واحد
- دمج شريط فلاتر المحلات + أزرار الترتيب في **شريط sticky خفيف** أعلى الشبكة (`sticky top-16 z-10 bg-background/95 backdrop-blur`) ليبقى متاحًا أثناء التمرير.
