

## ملء الفراغ السفلي بـ "شريط الثقة" (Trust Ribbon)

### تحديد الفراغ بدقة
الفراغ الذي تشير إليه يقع في **أسفل قسم "ابدأ من هنا" (Final CTA - القسم 12)** قبل قسم "لماذا مول البستان؟" (SeoIntroFooter الإيفوري). البطاقات الثلاث في CTA مضغوطة في عرض ضيق `max-w-[700px]` مع padding سفلي `clamp(56px, 7vw, 112px)`، مما يخلق شريطاً كحلياً عريضاً فارغاً بلا محتوى.

### الفكرة: شريط ختامي للثقة (Closing Trust Ribbon)

شريط أفقي رفيع داخل نفس القسم الكحلي (بلا قسم جديد، بلا تكرار للأفكار)، يحتوي على **4 شارات إثبات** أفقية متباعدة، تختم رحلة الزائر بأرقام موثوقة قبل التحوّل إلى الإيفوري.

```text
┌──────────────────────────────────────────────────────────────────────┐
│ ابدأ من هنا.                                                          │
│ [الخريطة] [دليل المحلات] [استفسار التأجير]                           │
│                                                                       │
│ ─────────────  فاصل رفيع متلاشي  ─────────────                       │
│                                                                       │
│  منذ 1990  ·  3 فروع  ·  رد خلال يوم عمل  ·  افتتاح 01 مايو 2026   │
└──────────────────────────────────────────────────────────────────────┘
```

### المحتوى التفصيلي

**4 شارات نصية متراصة أفقياً** (بدون أيقونات، بدون كروت، نص خالص فاخر):

| الشارة | النص | الدلالة |
|--------|------|---------|
| 1 | منذ 1990 | عمق السوق |
| 2 | 3 فروع نشطة | السعة |
| 3 | رد خلال يوم عمل | التزام تجاري |
| 4 | افتتاح 01 · 05 · 2026 | الحدث |

كل شارة: رقم/كلمة بارزة `#F1F5F9` + سطر صغير وصفي `#7C8BA1` تحته.

### التصميم البصري

- **الموقع**: داخل نفس قسم Final CTA، أسفل الـ grid مباشرة، مع `mt-10 md:mt-14`
- **الفاصل**: خط أفقي متلاشٍ `bg-gradient-to-r from-transparent via-white/8 to-transparent` بعرض كامل قبل الشارات
- **الترتيب**: `flex flex-wrap justify-center gap-x-8 md:gap-x-14 gap-y-4` للتوزيع الطبيعي
- **الفاصل بين الشارات**: نقطة دقيقة `·` بلون `#CDBB9A` فقط على الديسكتوب
- **الطباعة**: 
  - الرقم/الكلمة: `text-[0.85rem] md:text-[0.95rem] font-bold` بـ `#F1F5F9`
  - الوصف: `text-[0.62rem] tracking-[0.14em] uppercase` بـ `#7C8BA1` (إنجليزي بسيط أو عربي قصير)
- **بدون خلفية، بدون border، بدون كروت** — النص ينساب على الخلفية الكحلية القائمة
- **بدون إيموجي، بدون أيقونات Lucide** (التزام بسياسة المشروع)

### لماذا هذا الحل احترافي ولا يكرر شيئاً

1. **لا يكرر MallPulseStrip**: ذاك يقع في الأعلى (تحفيز للاستكشاف)، هذا يقع في الأسفل (ختم للثقة) — وظيفتان مختلفتان.
2. **لا يكرر WhyElBostan**: ذاك بطاقات أيقونية مع شرح؛ هذا شريط نصي خام بلا أيقونات ولا كروت.
3. **لا يكرر إحصائيات الأرقام**: في MallPulseStrip "150 محل / 3 أدوار / 10 تخصصات"؛ هنا "1990 / 3 فروع / SLA / تاريخ الافتتاح" — معلومات مختلفة تماماً (السنة، العدد المؤسسي، الالتزام، الحدث).
4. **يحترم رتم الصفحة**: لا قسم جديد، لا خلفية جديدة، فقط استثمار للمساحة الفارغة داخل القسم القائم.
5. **يحضّر الانتقال للإيفوري**: ينهي الكتلة الكحلية بنبرة هادئة واثقة قبل النص التحريري الإيفوري.

### التفاصيل التقنية

**التعديل الوحيد**: `src/components/home/HomeContent.tsx` — داخل القسم 12 (Final CTA)، بعد `</div>` الخاص بـ grid البطاقات الثلاث وقبل `</Reveal>`، إضافة blok جديد:

```tsx
{/* Closing Trust Ribbon — fills the navy gap before SEO footer */}
<div className="mt-10 md:mt-14">
  <div className="mx-auto h-px max-w-[28rem] bg-gradient-to-r from-transparent via-white/[0.08] to-transparent mb-6 md:mb-8" />
  <div className="flex flex-wrap items-baseline justify-center gap-x-8 md:gap-x-12 gap-y-4">
    {[
      { value: "منذ 1990", label: "ثقة السوق" },
      { value: "3 فروع نشطة", label: "حضور موسّع" },
      { value: "رد خلال يوم عمل", label: "التزام تجاري" },
      { value: "01 · 05 · 2026", label: "الافتتاح الرسمي" },
    ].map((item, i, arr) => (
      <div key={item.value} className="flex items-baseline gap-x-8 md:gap-x-12">
        <div className="text-center">
          <p className="text-[0.85rem] md:text-[0.95rem] font-bold leading-tight" style={{ color: "#F1F5F9", fontFamily: "var(--font-arabic-display)" }}>
            {item.value}
          </p>
          <p className="mt-1 text-[0.6rem] tracking-[0.14em]" style={{ color: "#7C8BA1" }}>
            {item.label}
          </p>
        </div>
        {i < arr.length - 1 && (
          <span className="hidden md:inline text-[0.6rem]" style={{ color: "#CDBB9A60" }}>·</span>
        )}
      </div>
    ))}
  </div>
</div>
```

**ملفات لن تُعدّل**:
- لا تعديل في `MallPulseStrip.tsx` (يبقى كما هو في الأعلى)
- لا تعديل في `WhyElBostan.tsx` أو `SeoIntroFooter.tsx`
- لا قسم جديد، لا ملف جديد، لا تغيير في الـ CSS أو tailwind config

### النتيجة

الشريط السفلي الكحلي الفارغ يتحوّل إلى ختم بصري احترافي مكوّن من 4 شارات ثقة أفقية، تنهي الرحلة بأسلوب "إيديتوريال" قبل الانتقال للنص التحريري الإيفوري. لا تكرار، لا حشو، فقط مساحة فارغة استُثمرت برسائل تجارية موزونة.
